# VariableStateButton
可以设置按下状态和非选中状态的alpha的按钮

```
 <com.qianfanyun.lib.VariableStateButton
        android:layout_width="match_parent"
        android:layout_height="50dp"
        app:color_background="#15bfff"
        app:unClick_alpha="0.6"
        app:pressed_alpha="0.6"
        app:bg_corner="5dp" />
```
